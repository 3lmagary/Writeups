# Hash3rror — Writeup بالعربي

## وصف التحدي

لدينا قيمة Hash تالفة تم الحصول عليها من ملف **PCAP**، ومعها ملاحظة تقول:

> **password = sha-1(hash-result)**

المعطى:

```
77be5d24ed2e3e590045e1d6o7e84i50d2799c19f48ede46804a8734e287df120f
```

## الفكرة العامة

المطلوب هو:

1. إصلاح الـ hash التالف.
    
2. معرفة نوع الخوارزمية المستخدمة.
    
3. فك التشفير للحصول على النص الأصلي.
    
4. عمل SHA‑1 للناتج للحصول على كلمة السر النهائية.
    

---

## الخطوات

### 1. إصلاح الـ Hash

الـ hash يحتوي حرفين غير صالحين في النظام الست عشري:

- **o**
    
- **i**
    

نقوم بإزالتهما:

```
77be5d24ed2e3e590045e1d67e8450d2799c19f48ede46804a8734e287df120f
```

هذا الآن Hash صالح.

### 2. تحديد نوع الـ Hash

استخدمنا أداة hash-identifier:

```bash
hash-identifier
```

وأعطتنا احتمالات:

- SHA‑256
    
- Haval‑256
    

### 3. فك التشفير

نقوم بتجربة تفكيك الـ hash على أحد مواقع كسر التجزئة مثل:

- md5decrypt.net
    

النتيجة:

```
77be5d24ed2e3e590045e1d67e8450d2799c19f48ede46804a8734e287df120f : s3cr3tpassword
```

إذن قيمة الـ hash تعود إلى:

```
s3cr3tpassword
```

### 4. تطبيق SHA‑1 على الناتج

طبقاً للملاحظة كلمة السر النهائية = SHA‑1 للناتج السابق.

نقوم بعمل ذلك باللينكس:

```bash
echo -n s3cr3tpassword | sha1sum
```
