# كلمة مرور Bandit27

**التاريخ:** 15 أكتوبر 2025

---

## الخطوات بالترتيب للحصول على كلمة مرور `bandit27`

1. **الاتصال بخادم Bandit عبر SSH**

   ```bash
   ssh bandit26@bandit.labs.overthewire.org -p 2220
   ```

2. **تعيين الـ shell إلى bash داخل محرر Vim (إن لزم)**

   داخل `vim` يمكنك تشغيل الأمر التالي لتحويل الـ shell إلى bash:

   ```vim
   :set shell=/bin/bash
   :shell
   ```

3. **استعراض محتويات المجلد الحالي**

   ```bash
   ls -al
   ```

   ستظهر لك ملفات مثل `bandit27-do` و `text.txt` مع أذونات ومالكين الملفات.

4. **تشغيل البرنامج ذو الصلاحية المسموح بها لقراءة ملف كلمة المرور**

   لاحظ وجود ملف تنفيذي باسم `bandit27-do` مملوك للمستخدم `bandit27` وله صلاحيات خاصة. لتشغيله واستخراج كلمة المرور نفّذ:

   ```bash
   ./bandit27-do cat /etc/bandit_pass/bandit27
   ```

5. **الاحتفاظ بكلمة المرور بأمان**

   بعد تشغيل الأمر ستظهر كلمة المرور مثل:

   ```text
   upsNCc7vzaRDx6oZC6GiR6ERwe1MowGB
   ```
